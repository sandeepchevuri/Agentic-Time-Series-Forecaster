### Features

* **Custom Forecaster Integration**: Users can now integrate specific models into the agent for evaluation and utilization through the `forecasters` argument in TimeCopilot's constructor. See [#153](https://github.com/TimeCopilot/timecopilot/pull/153).
    ```python
    from timecopilot import TimeCopilot
    from timecopilot.models.benchmarks import SeasonalNaive
    from timecopilot.models.foundational.toto import Toto

    tc = TimeCopilot(
        llm="openai:gpt-4o",
        forecasters=[
            SeasonalNaive(),
            Toto(),
        ]
    )
    tc.forecast(
        df="https://timecopilot.s3.amazonaws.com/public/data/air_passengers.csv", 
        h=12,
    )
    result = tc.query("What is the best model for monthly data?")
    print(result.output)
    ```
